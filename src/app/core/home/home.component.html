Welcome to Recipe Book App!


This is an Angular Recipe Book web application.

In this project, the mainly used technologies are listed below:

  • TypeScript(class, interface, enums etc)
  • Angular (component, ng-module, service, pipe, routing, directive, validation, unit tests etc)
  • ng-cli
  • Bootstrap 4
  • ng-bootstrap

Without signing in, you can only view the recipes and shopping list,
After Signing in, you can add/edit/delete the recipes and the shopping list.
